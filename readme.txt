在现有的 SPMA 接入算法中，尽管二进制指数退避（BEB）算法及其改进版本为数据传输提供了一定的公平性，但它们通常假设所有数据包都是平等的，这与 SPMA 协议划分不同优先级区分服务的设计目的相悖。基于优先级的退避策略虽然考虑了优先级序号、信道负载统计数据和阈值，但未能充分考虑系统整体性的影响。不同优先级用户在信道访问时可能相互影响，例如，高优先级用户的退避时间变化可能导致低优先级用户的延迟增加或分组投递率下降。这种相互影响如果不加以管理，可能会导致整个系统效率下降。
基于此，本章提出了一种基于 Q-learning 的 SPMA 退避方法。该方法的退避时间决策参考发送时刻的信道占用率、优先级序号以及各优先级阈值，综合考虑各级业务分组投递率与平均退避时延，对不同优先级的退避时间进行动态调整。通过利用 Q表，为不同优先级用户选择合适的退避时间，这将更好地适配SPMA 协议，在尽量保障高优先级业务的 QoS 下，提高低优先级业务的 QoS。
